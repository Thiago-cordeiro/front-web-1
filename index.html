Sobre o Projeto

Bem-vindo ao nosso projeto! Este site foi desenvolvido como parte do trabalho final da disciplina de Desenvolvimento Web 1 da Universidade Federal do Paraná. O objetivo deste trabalho foi integrar teoria e prática, criando uma aplicação completa que envolve:

  • Parte Visual (Front-end): Um design interativo e intuitivo que oferece uma experiência agradável para os usuários.
  • Parte Lógica (Back-end): A implementação da lógica de programação para realizar operações eficientes no sistema.
  • Armazenamento (Banco de Dados): Gerenciamento de dados em um banco de dados relacional, garantindo a persistência das informações.

Este site implementa um CRUD, que significa Create (Criar), Read (Ler), Update (Atualizar) e Delete (Deletar), que são operações fundamentais para gerenciar dados de maneira eficaz em qualquer sistema.

Estamos muito felizes com o resultado e ansiosos para mostrar o impacto que a tecnologia pode ter no processo de ensino e aprendizagem!
